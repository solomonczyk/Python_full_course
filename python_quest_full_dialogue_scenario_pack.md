# **Мини-игра: “Кубик судьбы”**

## Сюжет мини-игры

Герой выбирает один из двух магических кубиков. Кубик решает, сколько ходов получит герой.

## Что должна делать программа

1. Спросить номер кубика.

## Темы

- `if / elif / else`;
- `random.randint`;
- сравнения;
- переменные;
- базовые проверки;
- внимательность к условию.
